========================================================
 Claude Agent Orchestrator - how to run it
========================================================

WHAT YOU NEED (prerequisites)
-----------------------------
1) Node.js LTS        -> https://nodejs.org/
2) Claude Code CLI    -> in a terminal, run:
                         npm install -g @anthropic-ai/claude-code

.NET is NOT required - it is bundled inside.

HOW TO RUN
----------
1) Unzip this whole archive into a folder (e.g. C:\ClaudeOrchestrator).
2) Double-click "Start.bat".
   (Or run "ClaudeOrchestrator.exe" directly.)
3) Open your browser at:  http://localhost:6001

WHERE YOUR DATA LIVES
---------------------
The app stores its data in the "data" folder next to the program
(agents.json, priorities.json, history).

CHANGING THE PORT
-----------------
Edit "appsettings.json" -> the "Port" value,
or set the environment variable Port=8080.

TROUBLESHOOTING
---------------
* "Node.js is not installed" - install Node LTS (see above).
* Port 6001 in use - change the port (see above).
* Windows SmartScreen ("Windows protected your PC") - the program is
  not digitally signed. Click "More info" -> "Run anyway".
